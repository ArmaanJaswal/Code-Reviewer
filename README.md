# 🚀 AI Code Reviewer (Gemini API + React + Node.js + Express)

## 📌 Overview

This project is an **AI-powered Code Reviewer** that allows users to paste or upload code and receive instant, intelligent feedback. It leverages the **Gemini API** to analyze code for bugs, improvements, optimization, and best practices.

Built using a **React frontend** and a **Node.js + Express backend**, the application is lightweight and does not require a database.

---

## ✨ Features

* 🧠 AI-powered code analysis using Gemini API
* 🐞 Detects bugs and potential issues
* ⚡ Suggests optimizations and performance improvements
* 📚 Recommends best coding practices
* 💻 Clean and responsive user interface
* 🚀 Fast and lightweight (no database required)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/ArmaanJaswal/Code-Reviewer.git
cd Code-Reviewer
```

---

### 2️⃣ Setup Backend

```bash
cd backend
npm install
```

Create a `.env` file and add:

```env
GEMINI_API_KEY=your_api_key_here
```

Run backend:

```bash
npm start
```

---

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🚀 Usage

1. Open the frontend in your browser
2. Paste or upload your code
3. Submit the code for review
4. Get AI-generated feedback instantly
